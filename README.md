# grandpa

原子化脚手架工具。

## Usage

```bash
# package.json, .editorconfig
$ grandpa init
# tsconfig.json, build & dev script
$ grandpa g typescript
# .prettierrc, prettier and plugins
$ grandpa g prettier
# esno + zx
$ grandpa g script
# husky + lint-staged
$ grandpa g husky
$ grandpa g jest
$ grandpa g github-ci
$ grandpa g changelog
```
