# gimi

原子化脚手架工具。

## Usage

```bash
# package.json, .editorconfig
$ gimi init
# tsconfig.json, build & dev script
$ gimi g typescript
# .prettierrc, prettier and plugins
$ gimi g prettier
# esno + zx
$ gimi g script
# husky + lint-staged
$ gimi g husky
$ gimi g jest
$ gimi g github-ci
$ gimi g changelog
```
